Programming Basics

Variables & Data Types

Variables are containers for storing data values. Each variable has a specific type, which determines the size and layout of the variable's memory.

// Variable declaration and initialization
int age = 25;
double price = 19.99;
char grade = 'A';
bool is_valid = true;
std::string name = "Alice";

// Constants
const float PI = 3.14159;

Common Data Types:

  • int: Integer numbers (e.g., 42, -7, 0)
  • float/double: Floating-point numbers (e.g., 3.14, -0.001)
  • char: Single characters (e.g., 'A', '?', '\n')
  • bool: Boolean values (true/false)
  • string: Sequence of characters (e.g., "Hello")
Best Practice: Always initialize your variables when declaring them to avoid undefined behavior.

Operators

Operators perform operations on variables and values. C++ provides various types of operators:

Arithmetic Operators

int a = 10, b = 3;

cout << a + b << endl;  // Addition: 13
cout << a - b << endl;  // Subtraction: 7
cout << a * b << endl;  // Multiplication: 30
cout << a / b << endl;  // Division: 3 (integer division)
cout << a % b << endl;  // Modulus: 1 (remainder)

Relational Operators

cout << (a == b) << endl;  // Equal to: false (0)
cout << (a != b) << endl;  // Not equal to: true (1)
cout << (a > b) << endl;   // Greater than: true
cout << (a <= b) << endl;  // Less than or equal: false

Logical Operators

bool x = true, y = false;

cout << (x && y) << endl;  // AND: false
cout << (x || y) << endl;  // OR: true
cout << (!x) << endl;      // NOT: false

Conditional Statements

Conditional statements allow you to control program flow based on different conditions.

If-Else Statements

int score = 85;

if (score >= 90) {
    cout << "Grade: A" << endl;
} 
else if (score >= 80) {
    cout << "Grade: B" << endl;
} 
else if (score >= 70) {
    cout << "Grade: C" << endl;
} 
else {
    cout << "Grade: F" << endl;
}

Switch Statement

char grade = 'B';

switch (grade) {
    case 'A':
        cout << "Excellent!" << endl;
        break;
    case 'B':
        cout << "Good job!" << endl;
        break;
    case 'C':
        cout << "Average" << endl;
        break;
    default:
        cout << "Needs improvement" << endl;
}
Important: Always include break statements in switch cases to prevent fall-through behavior.

Loops

Loops execute code blocks repeatedly until a specified condition is met.

For Loop

// Print numbers 1 to 5
for (int i = 1; i <= 5; i++) {
    cout << i << " ";
}
// Output: 1 2 3 4 5

While Loop

// Countdown from 5
int count = 5;
while (count > 0) {
    cout << count << " ";
    count--;
}
// Output: 5 4 3 2 1

Do-While Loop

// Get user input at least once
int number;
do {
    cout << "Enter a positive number: ";
    cin >> number;
} while (number <= 0);
cout << "You entered: " << number << endl;

Functions

Functions allow you to group code into reusable blocks with defined inputs and outputs.

Function Definition

// Function declaration
int addNumbers(int a, int b) {
    return a + b;
}

int main() {
    int result = addNumbers(5, 7);
    cout << "Sum: " << result << endl;  // Output: Sum: 12
    return 0;
}

Parameters & Return Values

// Function with default parameter
double calculateArea(double radius, double pi = 3.14159) {
    return pi * radius * radius;
}

int main() {
    cout << calculateArea(5.0) << endl;        // Uses default pi
    cout << calculateArea(5.0, 3.14) << endl;  // Uses provided pi
    return 0;
}

Arrays

Arrays allow you to store multiple values of the same type in a single variable.

1D Arrays

// Initialize and access array elements
int numbers[5] = {10, 20, 30, 40, 50};

// Print all elements
for (int i = 0; i < 5; i++) {
    cout << numbers[i] << " ";
}

2D Arrays

// Matrix example (3x3)
int matrix[3][3] = {
    {1, 2, 3},
    {4, 5, 6},
    {7, 8, 9}
};

// Access elements
cout << matrix[1][2] << endl;  // Output: 6
